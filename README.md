# Currículo Digital - Barbara Costa Silverio

Este é um projeto de currículo digital desenvolvido em Angular, apresentando as informações profissionais de Barbara Costa Silverio de forma moderna e responsiva.

## 🚀 Tecnologias Utilizadas

- **Angular 20+** - Framework principal
- **TypeScript** - Linguagem de programação
- **SCSS** - Pré-processador CSS
- **Bootstrap 5** - Framework CSS para responsividade
- **Font Awesome** - Ícones

## 📋 Funcionalidades

- ✅ Design responsivo e moderno
- ✅ Seção de resumo profissional
- ✅ Experiência profissional detalhada
- ✅ Informações educacionais
- ✅ Habilidades técnicas com sistema de avaliação
- ✅ Informações de contato interativas
- ✅ Layout otimizado para impressão

## 🏃‍♂️ Como executar o projeto

### Pré-requisitos

- Node.js (versão 18 ou superior)
- npm ou yarn

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/babisilverio/babisilverio.github.io.git
cd babisilverio.github.io
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o projeto em modo de desenvolvimento:
```bash
npm start
```

4. Acesse http://localhost:4200 no seu navegador

## 📦 Build para produção

Para gerar os arquivos de produção:

```bash
npm run build
```

Os arquivos serão gerados na pasta `dist/`.

## 📁 Estrutura do projeto

```
src/
├── app/
│   ├── components/
│   │   ├── header/          # Cabeçalho com nome e cargo
│   │   ├── resumo/          # Resumo profissional
│   │   ├── experiencia/     # Experiências profissionais
│   │   ├── educacao/        # Formação acadêmica
│   │   └── contato/         # Informações de contato
│   ├── app.component.*      # Componente principal
│   └── app.module.ts        # Módulo principal
├── assets/                  # Recursos estáticos
├── styles.scss             # Estilos globais
└── index.html              # Página principal
```

## 🎨 Customização

Para personalizar o currículo com suas informações:

1. **Dados pessoais**: Edite os componentes em `src/app/components/`
2. **Cores e estilos**: Modifique `src/styles.scss` e os arquivos `.scss` dos componentes
3. **Layout**: Ajuste a estrutura em `app.component.html`
